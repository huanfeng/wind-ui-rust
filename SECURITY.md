# 安全政策 / Security Policy

## 支持的版本 / Supported versions

项目仍处于早期（0.x），仅对**最新发布版**提供安全修复。
The project is early-stage (0.x); security fixes target the **latest released version** only.

## 报告漏洞 / Reporting a vulnerability

**请勿**通过公开 issue 报告安全漏洞。
**Do not** report security vulnerabilities through public issues.

请用以下任一**私密**渠道 / Use one of these **private** channels:

- GitHub 私密漏洞报告 / GitHub private vulnerability reporting：
  仓库 **Security → Report a vulnerability**（[Advisories](https://github.com/huanfeng/wind-ui-rust/security/advisories/new)）
- 邮件 / Email：huanfengf@gmail.com

请尽量提供：受影响版本、复现步骤、影响面、可能的修复建议。
Please include: affected version, reproduction steps, impact, and any suggested fix.

## 响应 / Response

我们会在合理时间内确认并评估，确认后协调修复与披露时间。
We will acknowledge and assess within a reasonable time, then coordinate a fix and disclosure timeline.
作为个人维护的开源项目，无法承诺 SLA，但会尽力及时处理。
As a personal open-source project there is no guaranteed SLA, but reports are handled promptly on a best-effort basis.
